Questions

- What is the difference between a static and a dynamic web site?
A static website is fixed: server reads the html file then "sends" it to the client. A dynamic website is complicated: server runs a dynamic script then "outputs" and "sends" the generated html to the client. The current site is a static site.

- What is the main objective of using Domain Name System (DNS)?
Clients don't have to remember the IP addresses to access websites.

- Give some examples of generic top-Level domains (gTLD).
".com", ".edu", ".gov" ......

- Give some examples of country code top-Level domains (ccTLD).
".ca", ".jp", ".cn" ......

- Give an example of Uniform Resource Locator (URL) and specify all components of an URL.
"https://www.google.com/", "https" is the protocol; "www.google.com" is the domain name.

- What is the function (purpose) of Hypertext Transfer Protocol (HTTP)?
To provied a standard for web browsers (clients) and servers to talk to each other.

- Give some examples of HTTP request methods.
GET, HEAD, POST, PUT, DELETE ......

- Define a Web Server.
A computer stores server software and website's component files.
